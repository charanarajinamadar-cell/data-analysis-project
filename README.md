# 🧠 Sports Location Predictor (Machine Learning Project)

## 📌 Overview

This project predicts whether a sport is played **INDOOR** or **OUTDOOR** using a Machine Learning classification model based on NLP techniques.

---

## 🚀 Tech Stack

* Python
* Pandas
* Scikit-learn
* NLP (CountVectorizer)
* Multinomial Naive Bayes

---

## ⚙️ Workflow

* Created dataset of sports and their locations
* Converted text data into numerical format using CountVectorizer
* Trained model using Multinomial Naive Bayes
* Evaluated model accuracy using train-test split
* Predicted location for new sport input

---

## 📊 Example

Input: Cricket
Output: OUTDOOR

---

## ▶️ How to Run

Install dependencies:

```
pip install pandas scikit-learn
```

Run the project:

```
python sports_classifier.py
```

---

## 📈 Model Details

* Algorithm: Multinomial Naive Bayes
* Feature Extraction: CountVectorizer
* Type: Text Classification
